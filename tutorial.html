CSS 專屬
分類 語法 何時用 常見坑 示例
基本
.class / #id / div元件樣式、唯一元素、標籤統一過度依賴 id 造成難重用.btn { padding: 8px 12px; }
群組
h1, h2, h3同一套字級或間距把不相關元素綁一起h1,h2 { line-height:1.2; }
後代 / 子層
.card p / .list > li限制只在容器內生效選擇器層級太深.menu > li { list-style:none; }
兄弟關係
h2 + p / h2 ~ p控制標題後第一段或後續段落把 + 和 ~ 用反h2 + p { margin-top:6px; }
屬性選取器
input[type="email"]對特定型別精準套樣式選過廣導致非預期樣式a[target="_blank"]::after { content:"↗"; }
偽類（含第幾格）
:nth-child() / :nth-of-type()選第 N 個元素或奇偶列數錯索引（從 1 開始，不是 0）td:nth-child(3){font-weight:700;}
狀態偽類
:hover :focus :active互動狀態、鍵盤可及性只做 hover 不做 focusbutton:focus { outline:2px solid #4f46e5; }
偽元素
::before / ::after裝飾符號、標籤、分隔線忘記寫 content.tag::before{content:"NEW";}
CSS 變數
--brand / var(--brand)全站色彩、間距統一未設 fallback 值color: var(--brand, #4f46e5);
RWD 斷點
@media (min-width:640/768/1024px)手機→平板→桌機逐步增強只測桌機，手機爆版@media (min-width:768px){ .grid{grid-template-columns:repeat(2,1fr);} }
