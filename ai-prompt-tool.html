新手先這樣做：4 步完成第一份 Prompt

  1. 先選「應用分類」：教學 / 開發 / 行銷 / 影像...
  2. 再選一個「實戰情境」：例如 45 分工作坊或電商產圖
  3. 微調 Role/Task/Context/Style，看右側 Prompt 即時更新
  4. 點進階技巧（可複選），再複製 .md 直接貼到你常用 AI
